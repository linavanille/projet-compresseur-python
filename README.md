# Projet Compresseur – Huffman
Compresseur de données avec codage de Huffman

## Description
Ce projet consiste à implémenter un algorithme de compression de données basé sur le codage de Huffman.

L’objectif est de réduire la taille des fichiers en utilisant un codage binaire optimal basé sur la fréquence des caractères.


## Objectifs
- Comprendre les principes de la compression de données
- Implémenter l’algorithme de Huffman
- Encoder et décoder des données


## Fonctionnement

1. Analyse des fréquences des caractères
2. Construction d’un arbre de Huffman
3. Génération d’un code binaire optimal
4. Compression du fichier
5. Décompression possible


## Fonctionnalités
- Compression de texte
- Génération de codes binaires
- Décompression des données


## Technologies utilisées
- Python
- Structures de données (arbres, files de priorité)
- Algorithmique
